# NOW! Timer — 와이어프레임 명세서

> **버전**: v1.0  
> **캔버스 기준**: 2025년 작성  
> **업데이트 방법**: 캔버스에서 화면 수정 후 이 파일을 함께 업데이트해주세요.

---

## 색상 체계 (Color System)

| 용도 | 색상 | 설명 |
|------|------|------|
| 집중 모드 배경 | Grey (semi) | 편안한 중립 배경 |
| NOW! Lv.1 강조 | Yellow | 첫 번째 알림 신호 |
| NOW! Lv.2 강조 | Orange | 에스컬레이션 경고 |
| NOW! Lv.3 전체 배경 | Red (solid) | 최고 수준 강제 전환 |
| 핵심 기능 / 액션 버튼 | Blue | 확인·시작 버튼 등 |
| 통계 / 차트 | Orange | 데이터 시각화 |
| 소셜 상태 (정상) | Green | 집중 중 / 준수 상태 |
| 소셜 경고 | Orange / Red | NOW! 무시 상태 |

---

## 내비게이션 구조

하단 탭바 4개 탭 (모든 화면 공통):

```
[ 몰입 ]   [ 소셜 ]   [ 통계 ]   [ 설정 ]
```

- **몰입**: 타이머 집중 화면 (기본 진입점)
- **소셜**: 팀원 현황 & 알림
- **통계**: 세션 기록 & 반응 속도
- **설정**: 타이머 시간 / 알림음 / 팀 코드

## Row 1 — 핵심 플로우 화면 (y: 122–762)

> 핵심 플로우: 몰입 화면 → NOW! 알림(Lv.1) → NOW! 에스컬레이션(Lv.3)

---

### ① 몰입 화면 (Focus Mode)
**캔버스 ID**: `s1-bg`  
**배경색**: Grey (semi)

#### 구성 요소

| 요소 | 내용 | 캔버스 ID |
|------|------|-----------|
| 상단 앱바 | `NOW! Timer  ≡` (햄버거 메뉴) | `s1-topbar` |
| 중앙 집중 영역 | 빈 박스 (타이머 숫자 없음) | `s1-focus-zone` |
| 집중 텍스트 | `집중 중...` (중앙 큰 글씨) | `s1-focus-text` |
| 현재 작업명 | `프로젝트 기획서 작성` | `s1-task-name` |
| 진행 바 배경 | 얇은 회색 바 (시간 숫자 없음) | `s1-progress-bar-bg` |
| 진행 바 채움 | 파란 채움 (~60% 진행) | `s1-progress-bar-fill` |
| 타이머 숨김 안내 | `타이머 숨김 ON — 남은 시간 표시 없음` | `s1-progress-label` |
| 세션 카운터 | `세션 2 / 4  \|  오늘 목표 4개` | `s1-session-count` |
| 팀원 현황 미니 | 팀원 3명 상태 미리보기 (화스 / 수영 / 민준) | `s1-team-mini-bg` |
| 하단 내비 바 | `[ 몰입 ] [ 소셜 ] [ 통계 ] [ 설정 ]` | `s1-nav` |

#### 핵심 동작
- 타이머는 백그라운드에서 카운트다운, **숫자는 화면에 표시 안 됨**
- 진행 바만 얇게 표시 (시간 감각 최소화)
- 세션 종료 시 → 화면 ②로 전환

### ② NOW! 알림 — Level 1 (첫 신호)
**캔버스 ID**: `s2-bg`  
**배경색**: Grey (semi) + Yellow 글로우 효과 (ellipse)

#### 구성 요소

| 요소 | 내용 | 캔버스 ID |
|------|------|-----------|
| 상단 앱바 | `NOW! Timer` | `s2-topbar` |
| 노란 글로우 원 | 중앙 배경 발광 효과 | `s2-glow` |
| 메인 텍스트 | `NOW!` (크고 굵게) | `s2-now-text` |
| 서브 텍스트 | `쉴 시간이에요` | `s2-sub-text` |
| 알림음 배지 | `♪  Bell 재생 중` | `s2-sound-badge` |
| 확인 버튼 | `✓  확인 (쉬러 가기)` (파란 버튼) | `s2-dismiss-btn` |
| 스누즈 버튼 | `5분 후에...` (회색 보조 버튼) | `s2-snooze-btn` |
| 에스컬레이션 안내 | `* 무시하면 → Lv.2 → Lv.3으로 자동 에스컬레이션` | `s2-escalation-note` |
| 에스컬레이션 강도 | `에스컬레이션 강도:  [▓░░]  Lv.1 / 3` | `s2-escalation-bars` |
| 하단 내비 바 | `[ 몰입 ] [ 소셜 ] [ 통계 ] [ 설정 ]` | `s2-nav` |

#### 핵심 동작
- 확인 → 휴식 모드 진입
- 5분 후에 → 5분 뒤에 다시 NOW! Lv.1
- 무시(30초 경과) → 자동으로 Lv.2로 에스컬레이션

## 플로우 요약 (캔버스 플로우차트 참조)

```
START
  └─→ 집중 모드 (타이머 숨김)
        └─→ [세션 종료?]
              ├─ NO → 계속 집중
              └─ YES → NOW! Lv.1 발생 (소리 + 팝업)
                    └─→ [30초 내 반응?]
                          ├─ YES → 휴식 모드 시작
                          │         └─→ [휴식 종료?]
                          │               └─ YES → NOW! 복귀 신호 Lv.1
                          │                         └─→ [1분 내 복귀?]
                          │                               ├─ YES → 다음 세션 (루프)
                          │                               └─ NO → 에스컬레이션 → 친구 알림
                          └─ NO → NOW! Lv.2 (더 강한 소리 + 화면 흔들림)
                                └─→ [1분 내 반응?]
                                      ├─ YES → 휴식 모드 시작
                                      └─ NO → NOW! Lv.3 (전체 화면 빨강 점멸)
                                            └─→ [반응함?]
                                                  ├─ YES → 휴식 모드 시작
                                                  └─ NO → 친구들에게 자동 푸시 알림
```

### 소셜 알림 조건
| 단계 | 동작 |
|------|------|
| Lv.2 진입 시 | 팀원 목록에 경고 상태 표시 |
| Lv.3 진입 시 | 팀원들에게 자동 브라우저 푸시 알림 발송 |
| 팀원이 직접 | 흔들어 깨우기 버튼으로 1:1 푸시 알림 전송 |
